Bienvenido profesor a mi proyecto

mi trabajo se centra mas en la forma de aplicar la base de datos junto con python en un ambiente laborañ, por eso decidi crear esta paguina mas enfocado en lo laboral y no en un portafolios personal.

¿como lo realize?

Por supuesto este trabajo se realizo principalmente por chatgpt, pero este mismo genera el codigo con un problema grave, la dificultad de hacer cambios despues de agregar alguna opcion y implementar herramientas como mysql o puTTY.

proceso de realizacion de este proyecto

  • pensar en la idea y planeacion de proyecto –-- creacion del html y css como base
  • pedirle a chatgpt el python y javascript –---- arreglar la cantidad inmensa de errores
  • creacion de base de datos con xampp –--- cambio por mysql por comodidad
  • traslado a railwey (chatgpt no ayudo mucho) –--- corregir errores graves y quedo bien

¿que aprendi de todo esto?

Aprendi que chatgpt funciona hasta cierto punto, pero sirve como base para cualquier proyecto. Aunque ya tenia una base, aprendi mucho de mysql, python, puTTY, entre otras herramientas.
